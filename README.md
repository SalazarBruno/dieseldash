# DieselDash v1.0
### Universal 4-Cylinder Diesel EGT & AFR Monitor

[![License: GPL v3](https://shields.io)](https://gnu.org)

A Arduino-based monitoring system for diesel engines. This project integrates 4-channel EGT (Exhaust Gas Temperature) and real-time Wideband AFR (Air-Fuel Ratio) monitoring into a single dashboard.

**[Leia este documento em Português (Brasil)](README.pt-br.md)**

## 🚀 Features
- **4-Channel EGT:** Monitor individual cylinder health.
- **Wideband AFR:** Track combustion efficiency and smoke levels.
- **Peak Hold:** Memory mode to review maximum temperatures.
- **Error Detection:** Detects disconnected sensors (displays "ERR").
- **Dual Alarms:** 
  - **High EGT:** Fast high-pitched beep + flashing LED.
  - **Rich/Smoke:** Low-pitched growl + solid LED.

## 🛠 Hardware Support
DieselDash is modular. It currently supports:
- **EGT:** Innovate TC-4, FuelTech ETM-1, AEM X-Series, PLX Devices.
- **AFR:** Innovate LC-1/LC-2, AEM X-Series.
