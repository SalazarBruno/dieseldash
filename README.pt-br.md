# DieselDash v1.0 (Português)
### Monitor Universal de EGT e AFR para Motores Diesel de 4 Cilindros

Este é um sistema de monitoramento robusto baseado em Arduino. O projeto integra 4 canais de EGT (Temperatura dos Gases de Escape) e monitoramento de AFR (Mistura Ar-Combustível) em tempo real.

**[Read this document in English](README.md)**

## 🚀 Funcionalidades
- **EGT de 4 Canais:** Monitore a saúde de cada cilindro individualmente.
- **Wideband AFR:** Acompanhe a eficiência da combustão e níveis de fumaça.
- **Memória de Pico:** Modo para revisar as temperaturas máximas atingidas.
- **Detecção de Erro:** Identifica sensores desconectados (exibe "ERR").
- **Alarmes Duplos:** 
  - **EGT Alta:** Bipe agudo rápido + LED piscando.
  - **Mistura Rica:** Som grave + LED sólido.

## 🛠 Hardware Suportado
O DieselDash é modular e suporta nativamente:
- **EGT:** Innovate TC-4, FuelTech ETM-1, AEM X-Series, PLX Devices.
- **AFR:** Innovate LC-1/LC-2, AEM X-Series.
